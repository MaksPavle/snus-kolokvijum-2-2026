# snus-kolokvijum-2-2026

## Description
Project simulates "chaotic cupid" who sends random love messages to all online users. 
It is based on publish-subscribe model, where users login through a client, and then the server _(cupid)_ sends a random message from other users every minute they're online. They can block other users and they cannot get more messages before they confirm they've read the last one they got.
